
## File Searcher
A desktop application for searching files by name and content with advanced filtering options.

### System Requirements
* Java 8 or higher
* Windows 7+, macOS 10.10+, or Linux
* **Conflict-Free Orchestration:** A centralized manager system handles all core interactions (placing, breaking, rotating, inventory) to ensure zero conflicts between concurrently running modules.

### Installation
* **1** Install Java (if not already installed):
* **2** Verify Java installation:
        Open Command Prompt (Windows) or Terminal (macOS/Linux)
        Type: java -version
        Confirm you see version information
### Running the Application
* **1** Download FileSearcher.jar
* **2** Double-click the file
* **3** If prompted, select "Open with Java(TM) Platform SE Binary"
